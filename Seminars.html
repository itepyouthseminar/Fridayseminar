14.10

  • Надежда Костюченко. Оператор градиента как векторный оператор.

    Для всякой скалярной величины, заданной в виде функции пространственных координат, тройка ее производных по координатам в каждой точке просранства образует вектор, который имеет простой геометрический смысл: он указывает направление наискорейшего изменения функции в данной точке и по модулю равен величине этого изменения. Этот вектор называется градиентом функции.
